Atributo Descripción Explicación Ejemplo
background-color establece el color de fondo de un elemento. Se puede especificar el color como un valor RGB hexa-decimal o regular .exampletwo {
background-color: rgb(153,102,153);
}
background-image define la imagen de fondo de un elemento. Localización de la imagen que se utilizará de fondo. div {
background-image: url("https://mdn.mozillademos.org/files/logo.png");
}
background-repeat especifica si la imagen es repetida (mosaïco) y de que manera. repeat, -x, -y horizontal y vertical .two {
background-repeat: repeat;
}
font-size especifica la dimensión de la letra. Segun el % que pongamos será mas o menos grante la letra. .percent {
font-size: 200%;
}
font-weight especifica el peso o grueso de la letra Segun el numero que pongamos la letra será mas o menos gruesa. div {
font-weight: 600;
}
font-family define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado fantasy es un tipo de fuente. .examplefantasy {
font-family: fantasy;
}
text-align alinea el texto podemos usar left, right, center, justify... .right {
text-align: right;
}
text-transform Se puede utilizar para hacer que el texto aparezca en todo en mayúsculas o en minúsculas text-transform : uppercase ;
border permite definir de un golpe todos los bordes en una única regla de la hoja de estilos. prodemos utilizar propiedades border-width, -style y -color .black {
border: 1px solid #000;
}
list-style-type especifica la apariencia del listado. ol.normal {
list-style-type: upper-alpha;
}
text-decoration se usa para decorar el texto como por ejemplo subrrayado. h1.under {
text-decoration: underline;
}
padding establece el espacio de relleno requerido por todos los lados de un elemento Los px que podemos poner pueden ser entre 1 y 4, cada numero se aplica a un lado. h4{
background-color: green;
padding: 50px 20px 20px 50px;
}
margin establece el margen para los cuatro lados Los px que podemos poner pueden ser entre 1 y 4, cada numero se aplica a un lado .ex2 {
margin: 20px 0px 0 -20px;
}
